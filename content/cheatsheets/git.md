---
title: Git
---

Log all commits:
```
$ git log
```

Log last commit:
```
$ git log -1
```

Log last commit + patch output:
```
$ git log -p -2
```

Create branch:
```
$ git pull
$ git checkout -b branch-name
```

Merge branch:
```
$ git checkout master
$ git merge branch-name
$ git commit -m
$ git push origin master
```
