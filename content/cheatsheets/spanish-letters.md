#### Las letras españolas

How to type Spanish accents, Spanish letters, and special characters.



**Tildes:**

**Opt + e, then a = á**

**Opt + e, then e = é**

**Opt + e, then i = í**

**Opt + e, then o = ó**

**Opt + e, then u = ú**



**Eñe:**

**Opt + n, then n = ñ**



**Umlaut:**

**Opt + u, then u = ü**



**Signos de apertura:**

**Opt + 1 = ¡**

**Opt + shift + ? = ¿**
